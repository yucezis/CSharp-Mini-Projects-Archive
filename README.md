Welcome to my backend development portfolio! This repository is a comprehensive collection of 19 distinct projects, demonstrating my proficiency in the modern **.NET ecosystem**, various data management technologies, and architectural approaches.

As a Computer Engineering student and an aspiring .NET Backend Developer, my primary goal with this series is to build robust, scalable, and secure backend architectures by diving deep into relational and NoSQL databases, ORM strategies, API integrations, and security protocols (JWT).
